# 손약사 상담창구

## 프로젝트 설명
24시간 기본 약학 상담을 해주는 웹페이지입니다.

## 사용 기술
- HTML
- CSS
- JavaScript

## 파일 구조
```
260207/
├── index.html          # 메인 페이지
├── styles.css          # 스타일시트
├── script.js           # JavaScript 로직
└── CLAUDE.md          # 프로젝트 문서
```

## 실행 방법
1. 브라우저에서 `index.html` 파일을 엽니다.
2. 별도의 서버 설치나 설정 없이 바로 실행 가능합니다.

## 코딩 규칙
1. **한국어 주석 사용**: 모든 주석은 한국어로 작성합니다.
2. **초보자 친화적 코드**: 초보자가 이해하기 쉽게 코드를 작성합니다.
3. **설명 주석 필수**: 각 코드 블록마다 설명 주석을 작성합니다.
4. **명확한 변수명**: 변수와 함수명은 그 역할을 명확히 알 수 있도록 작성합니다.

## 개발 가이드라인
- 코드 작성 시 각 기능별로 주석을 상세히 작성합니다.
- 복잡한 로직은 단계별로 나누어 설명합니다.
- 사용자 인터페이스는 직관적이고 이해하기 쉽게 구성합니다.
